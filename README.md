# 10 Academy: Artificial Intelligence Mastery - Week 1 Challenge

## Project Overview
This repository hosts the code and analysis for weekly AI challenges, focused on financial news and stock data analysis.

## Task 1: Git and GitHub
### Usage
- Create a GitHub repository to host all code
- Create a branch  for analysis
- Perform Exploratory Data Analysis (EDA):
  - Descriptive Statistics
  - Text Analysis (Topic Modeling)
  - Time Series Analysis
  - Publisher Analysis
    
## Task 2: Quantitative Analysis using PyNance and TA-Lib
### Usage
- Prepare data including columns: Open, High, Low, Close, Volume
- Calculate technical indicators:
  - Moving averages
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
- Create visualizations to understand stock price behavior
