# 10 Academy: Artificial Intelligence Mastery - Week 1 Challenge

## Project: Predicting Price Moves with News Sentiment

Analyze real-world financial news to understand how headlines influence stock movements. Perform sentiment analysis, compute technical indicators, and correlate news sentiment with daily stock returns.

## Task 1

- Perform Exploratory Data Analysis (EDA) on text and time series data
- Run sentiment analysis on news headlines

## Dataset

- `headline`: News title
- `url`: Link to full article
- `publisher`: News source
- `date`: Publication date and time (UTC-4)
